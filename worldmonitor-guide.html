What is WorldMonitor

WorldMonitor is a free, open source global intelligence dashboard. It pulls live data from WHO, NASA, IMF, IAEA and 65+ other sources to give you real-time visibility into world conflicts, oil & gas pipelines, energy crises, Bitcoin prices, country instability scores, and breaking news — all in one place, running locally on your laptop.

What You Can Monitor
🌍
World Conflicts & Risks Country instability scores, active conflict zones, military activity
⛽

📰

Install in 4 Steps
1
Clone the Repository
Open your terminal and run this command to download the project to your laptop.
git clone https://github.com/koala73/worldmonitor.git
2
Go Into the Project Folder
Navigate into the downloaded folder.
cd worldmonitor
3
Install Dependencies
This installs everything the project needs. Wait for it to finish — takes 1-2 minutes.
npm install
4
Run the Dashboard
Start the app. You will see a local URL appear in the terminal.
npm run dev
Before You Start
Prerequisites: Make sure you have Node.js installed on your laptop before running these commands. Download it free from nodejs.org if you don't have it already. No API keys, no account, no payment required.
