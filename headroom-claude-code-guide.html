Why It Works

What Headroom Does

Claude Code burns tokens reading tool outputs, file contents, and long conversation history. Headroom sits between your agent and the LLM, compressing all of that using 6 algorithms — JSON, AST, text, image — before it's sent. Your data stays local. The model gets the same signal at a fraction of the size.

Step-by-Step Setup

Set Up Headroom in Claude Code

1
Step 1
Check Your Python Version

Headroom requires Python 3.10+. On macOS the system Python (3.9) is too old — install 3.13 from python.org if needed.

python3 --version
# Need 3.10 or higher
# macOS: install Python 3.13 from python.org if below 3.10
2
Step 2
Install Headroom

Install with the [all] extra to get every compressor, the proxy, MCP server, and memory features.

pip install "headroom-ai[all]"

# On macOS with Python 3.13 specifically:
/usr/local/bin/python3.13 -m pip install "headroom-ai[all]"

# With pipx (recommended for global CLI tools):
pipx install --python python3.13 "headroom-ai[all]"
3
Step 3
Add Headroom to Your PATH

The headroom CLI installs into Python's bin directory. Add it so you can run it from anywhere.

# Add to ~/.zshrc or ~/.bashrc:
export PATH="/Library/Frameworks/Python.framework/Versions/3.13/bin:$PATH"

# Reload your shell:
source ~/.zshrc

# Verify:
headroom --version
4
Step 4 — Main Step
Wrap Claude Code

One command starts the Headroom proxy and launches Claude Code through it. Token compression happens automatically — no config needed.

headroom wrap claude

# With cross-agent memory:
headroom wrap claude --memory

# With code graph (better code search compression):
headroom wrap claude --memory --code-graph

# For 1M context window:
headroom wrap claude --1m
What this does: Headroom starts a local proxy on port 8787, sets ANTHROPIC_BASE_URL through it, then launches Claude Code. Prompts, tool outputs, and file reads are compressed before reaching Anthropic's API. Your data never leaves your machine.
5
Step 5 Optional
Install as MCP Server

Adds Headroom as a native MCP tool inside Claude Code — unlocks headroom_compress, headroom_retrieve, and headroom_stats.

headroom mcp install

# Verify it registered:
claude mcp list
6
Step 6 Optional
Enable Output Token Reduction

Cuts what Claude writes back too. On Opus-class models output costs 5x input — this adds up fast.

# Add to ~/.zshrc:
export HEADROOM_OUTPUT_SHAPER=1

# Let Headroom learn your preferred verbosity from past sessions:
headroom learn --verbosity
headroom learn --verbosity --apply
7
Step 7
Check Your Savings

After a few sessions, see exactly how many tokens you've saved.

headroom perf
headroom dashboard
headroom output-savings
