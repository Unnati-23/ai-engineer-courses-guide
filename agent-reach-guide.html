What is this?

AGENT REACH

Agent Reach is an installer and router for internet access tools. It gives Claude Code the ability to search and read from 13 real-world platforms — all from a single setup, without switching apps.

Instead of copying content manually, Claude pulls live data directly while you work.

Before You Start

PREREQUISITES

Make sure you have all of these before running any commands:

  • Mac computer — this guide is macOS only
  • Python 3 — check with python3 --version in Terminal
  • Google Chrome — needed for the OpenCLI browser extension
  • GitHub account — free at github.com
  • Groq account — free at console.groq.com (for podcast transcription)
01

INSTALL AGENT REACH

Open Terminal and run these commands in order. This creates a Python virtual environment and installs Agent Reach with all channel backends.

Create the virtual environment

bash
python3 -m venv ~/.agent-reach-venv

Install Agent Reach

bash
source ~/.agent-reach-venv/bin/activate && pip install https://github.com/Panniantong/agent-reach/archive/main.zip

Run the auto-installer

bash
agent-reach install --env=auto

Install all optional channels

bash
agent-reach install --channels=all
Expected output
Installation complete! 5/13 channels active. (More will unlock in the next steps.)
02

FIX YOUTUBE

This one-liner configures yt-dlp with the correct JavaScript runtime so YouTube video info and subtitles work correctly.

bash
mkdir -p "$HOME/Library/Application Support/yt-dlp" && grep -qxF -- '--js-runtimes node' "$HOME/Library/Application Support/yt-dlp/config" 2>/dev/null || printf '%s\n' '--js-runtimes node' >> "$HOME/Library/Application Support/yt-dlp/config"
Note
Safe to run multiple times — it won't add duplicate lines.
03

AUTHENTICATE GITHUB

This connects the gh CLI to your GitHub account so Claude can read repos, issues, PRs, and code.

bash
gh auth login

When the interactive menu appears, select:

  • Where do you use GitHub? → GitHub.com
  • Preferred protocol? → HTTPS
  • Authenticate Git with your GitHub credentials? → Y
  • How would you like to authenticate? → Login with a web browser
  • Copy the one-time code shown in Terminal → paste it in the browser page that opens → click Authorize
You'll see
✓ Authentication complete. Logged in as YOUR_USERNAME
04

INSTALL OPENCLI CHROME EXTENSION

OpenCLI bridges Claude to your Chrome browser session. This unlocks Reddit, XiaoHongShu, LinkedIn, and Twitter — no separate login or API key needed.

Step 1 — Install the extension in Chrome

Chrome Web Store Link
Open Chrome and go to:
chromewebstore.google.com → search "OpenCLI" → Add to Chrome →

Step 2 — Restart the daemon

bash
source ~/.agent-reach-venv/bin/activate && opencli daemon restart

Step 3 — Verify the connection

bash
opencli doctor
You'll see
[OK] Daemon: running
[OK] Extension: connected
[OK] Connectivity: connected
05

ADD GROQ API KEY

Groq powers podcast and audio transcription. It's completely free — takes under 2 minutes to set up.

  • Go to console.groq.com and create a free account
  • Click API Keys → Create API Key
  • Copy the key (it starts with gsk_)
  • Paste it into the command below and run it in Terminal
bash
source ~/.agent-reach-venv/bin/activate && agent-reach configure groq-key gsk_YOUR_KEY_HERE
⚠ Security
Never share your API key publicly or paste it in chat. If you accidentally expose it, delete and regenerate at console.groq.com immediately.
06

VERIFY EVERYTHING

Run this final check to confirm all channels are active:

bash
source ~/.agent-reach-venv/bin/activate && agent-reach doctor
Expected result
✅ Installation complete! 11/13 channels active.
