Before You Start

API Keys — What You Actually Need

Only Key 01 is always required for every agent — it's what powers the agent on Claude's servers. Keys 02 and 03 are needed specifically because of what I wanted my agent to do — write to Notion and scan Product Hunt. If you build a different agent (email digest, Slack summary, etc.), you'll need different keys depending on what your agent connects to.

KEY 01 · ALWAYS NEEDED
Anthropic API Key
Every agent needs this — it's what runs your agent on Claude's cloud infrastructure, on a schedule, even when your laptop is off. This is separate from your Claude Code subscription.
console.anthropic.com → API Keys
Go to the Anthropic Console (not platform.claude.com) → create a new key → add some credit to your account so the agent can make API calls.
Required for every agent
KEY 02 · FOR MY AGENT
Notion Access Token
I wanted my agent to write the daily briefing to a Notion page — so it needs permission to access my Notion workspace. If you don't use Notion, you don't need this.
notion.so/profile/integrations
Create a Personal Access Token → create a Notion page called "AI Morning Brief" → share the page with your integration via Connections. Copy the page URL — the agent will ask for it.
Only if your agent uses Notion
KEY 03 · FOR MY AGENT
Product Hunt API Key + Secret
I wanted my agent to scan Product Hunt for new AI launches — so it needs API access. GitHub and HuggingFace are public and need no key. Only Product Hunt requires one.
api.producthunt.com → API Dashboard
Click "Add an Application" → Name it anything → Redirect URI: any valid HTTPS URL → you get a Client ID and Client Secret.
Only if your agent scans Product Hunt
The Exact Setup

5 Steps. ~5 Minutes.

You only need Claude Code installed and signed in, and your 3 API keys ready.

1
Open the Claude Code Chat Window
Fire up Claude Code on your MacBook (you need a Claude Pro or Max subscription). This is where the whole thing happens — no separate terminal, no extra setup.
Claude Code → Chat mode
2
Paste the GitHub Repo + Install Prompt
Drop in the repo link, then this exact prompt:
# github.com/anthropics/launch-your-agent
Install this skill globally and open the text file for me to paste the API key.
Claude Code will clone the repo, install the skill globally into ~/.claude/skills/, and open the .env file automatically.
3
Paste Your Anthropic API Key into the .env File
A text file opens — it has a line that says ANTHROPIC_API_KEY=. Paste your key directly after the equals sign. Press Cmd + S to save.

⚠ Important: Paste the key directly into the file — do NOT type it in the Claude Code chat. This keeps it private and secure.
Go to platform.claude.com → API Keys to get your key
4
Create Your Notion Page
Go to app.notion.com → click + Add new in the sidebar → name the page "AI Morning Brief".

Then click ... (three dots) at the top right of the page → Connections → connect your Notion integration.

Copy the page URL from the address bar — you'll paste this when the agent asks for the Notion target.
5
Run /launch-your-agent and Describe Your Agent
In the Claude Code chat, type:
/launch-your-agent
The skill starts an interview. Answer each question:

→ What should the agent do? → paste the prompt below
→ Timezone? → say India / IST / Asia/Kolkata
→ Notion token? → paste your secret_... token
→ Notion page URL? → paste the URL you copied
→ Product Hunt keys? → paste Client ID and Secret
The agent builds itself, runs a test, grades the output, and sets up a scheduled deployment — all automatically.
The Exact Prompts

Copy nothing. Understand everything.

Two things to run — in this order. First the repo to install the skill. Then the slash command with your agent description.

Step 1 — Paste This First in Claude Code
Repo + Install Prompt
github.com/anthropics/launch-your-agent

Install this skill globally and open the text file for me to paste the API key.
This installs the skill, clones the repo into ~/.claude/skills/, and opens the .env file so you can paste your Anthropic API key. Save with Cmd + S.
Step 2 — Then Run This + Describe Your Agent
/launch-your-agent + Agent Prompt — @aiwithunnati
/launch-your-agent
When it asks what your agent should do — paste this:
Scan GitHub trending repos, HuggingFace new model releases, and Product Hunt AI launches from the last 24 hours.

Pick the top 5, explain why each one matters for AI engineers, and write the full briefing to my Notion workspace every morning at 6 AM.
This is my agent's prompt — you can replace it with anything. Meeting briefs, inbox triage, competitor tracking. The slash command stays the same. Only your description changes.
