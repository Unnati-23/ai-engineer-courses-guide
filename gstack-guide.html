Installation Guide

From Zero To GStack In 4 Steps

Follow every command below. Each one is exactly what we ran — copy and paste directly into your terminal.

1
Install Claude Code

Claude Code is Anthropic's official CLI. Install it globally via npm. If you already have it, skip to Step 2.

Terminal 
npm install -g @anthropic-ai/claude-code

Verify it installed correctly:

Terminal 
claude --version

You should see something like 2.1.183 (Claude Code)

⚡ Note You need Node.js installed first. Get it at nodejs.org if you don't have it.
2
Verify Git Is Installed

GStack clones from GitHub, so you need Git. Check if it's already on your machine:

Terminal 
git --version

Expected output: git version 2.39.5 (Apple Git-154)

⚡ Mac Users If Git is missing, run xcode-select --install — it installs Git automatically.
3
Install Bun (JavaScript Runtime)

GStack uses Bun to build its browser binary and compile skills. Install it with the official one-liner:

Terminal 
curl -fsSL https://bun.sh/install | bash

After it installs, reload your shell so the PATH updates:

Terminal 
exec /bin/zsh

Verify Bun is ready:

Terminal 
bun --version

You should see: 1.3.14

4
Clone & Install GStack

This is the main event. One command clones Gary Tan's GStack repo directly into your Claude skills folder and runs the full setup — including downloading Playwright (headless Chrome) and compiling all 23 tools.

Terminal — this takes ~1-2 min 
git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git ~/.claude/skills/gstack && cd ~/.claude/skills/gstack && ./setup
⚡ What happens during setup Bun installs 297 packages → 55+ skill files generated → Playwright Chromium (~162 MB) downloaded → browse binary compiled. This only runs once.

The GStack repo lives at: github.com/garrytan/gstack
