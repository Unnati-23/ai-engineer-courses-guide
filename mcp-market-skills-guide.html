BEFORE YOU START

What's an Agent Skill?

A Skill is a small folder built around one file — SKILL.md — that teaches Claude how to do a specific job: write landing page copy, run an SEO audit, format a spreadsheet, whatever you need. Claude reads it and decides on its own when to use it, so you never have to re-explain the task in every chat.

MCP Market is a free, community-run directory of these Skills for Claude, Codex, and other AI agents — it isn't an official Anthropic product, just a catalogue. Anyone can browse it and download what they need.

THE PROCESS

5 steps, start to finish

01

Find a Skill on MCP Market

Go to mcpmarket.com/tools/skills and search for whatever task you want help with — "landing page popup," "SEO audit," "PDF," anything. Open the one you like and hit download.

You'll get a folder (sometimes zipped) with a SKILL.md file inside. That's the whole Skill.

02

Add it to Claude.ai (the web app)

  • Click your profile icon (top right) → Settings
  • Go to Settings → Capabilities and make sure "Code execution and file creation" is turned on — Skills need this to run
  • Go to Customize → Skills
  • Click the "+" button → Create skill, and upload the file you downloaded
  • Toggle the Skill on — it now shows up in your Skills list
claude.ai → Settings → Customize → Skills
03

Add it to Claude Code (if you use the CLI)

Unzip the folder and drop it into .claude/skills/ inside your project — that makes it available just there. Drop it into ~/.claude/skills/ instead, and it's available across every project on your machine.

Claude Code picks it up automatically. No restart needed for most changes.

04

Using Codex, Cursor, or another agent?

Skills follow an open format (agentskills.io), so most agent tools now read the same SKILL.md structure — only the folder location changes per tool.

Codexper-agent skills folder, check Codex docs for the exact path
Cursor / Gemini CLIsame SKILL.md format, tool-specific skills directory
05

Just describe the task — that's it

You don't "run" a Skill manually. Type your request normally, and Claude reads the Skill's description in the background and decides if it's relevant. If it is, it uses it without you asking.

Before you install anything

Skills can include executable code, not just instructions — so treat them like installing software. Only download from sources you trust, and skim the SKILL.md file before uploading it.
